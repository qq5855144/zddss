# 项目名称
基于 React 的企业级管理后台解决方案

## 特性
- 开箱即用的权限管理系统
- 响应式布局适配多端
- 集成常用业务组件

## 快速开始
### 安装依赖
```bash
npm install
# 或
yarn install
```

### 开发模式
```bash
npm run dev
# 或
yarn dev
```

### 生产构建
```bash
npm run build
# 或
yarn build
```

## 技术栈
- React 18
- Redux Toolkit
- Ant Design 5.x
- Vite 4.0

## 贡献指南
1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feat/xxx`)
3. 提交修改 (`git commit -m 'feat: 新增xxx功能'`)
4. 推送分支 (`git push origin feat/xxx`)
5. 创建 PR
